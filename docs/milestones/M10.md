# M10 ‚Äî Notifications & Communication ‚Äî ‚úÖ STEPS 1‚Äì10 COMPLETE (awaiting approval)

Scope: a complete **in-app** notification system over frozen M1‚ÄìM9 ‚Äî a `Notification` event + explicit
per-user `NotificationRecipient` copies, generated by business services **after** a publish commits; a bell +
unread badge + inbox on mobile and web; read/archive tracking; deep links; and an admin announcement composer.
Purely **additive** (2 new tables, 2 enums, permission-only). **NO push / SMS / WhatsApp / email / chat.**
**ADR-018** governs the design.
